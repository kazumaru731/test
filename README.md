# あみだくじアプリ

自分以外の人に必ず当たるあみだくじを作成するWebアプリケーションです。

## 特徴

- **完全順列アルゴリズム**: 数学的に全員が自分以外の人に必ず当たるように設計されています
- **視覚的なあみだくじ**: Canvas APIを使用したきれいなあみだくじの描画
- **シンプルなUI**: 直感的で使いやすいインターフェース

## 使い方

1. `index.html` をブラウザで開く
2. 参加者の名前を入力して「追加」ボタンをクリック（最低2人必要）
3. 参加者を追加し終えたら「あみだくじを開始」をクリック
4. あみだくじの結果が表示されます

## 技術仕様

### 完全順列（Derangement）

このアプリは「完全順列」という数学的概念を使用しています。完全順列とは、順列の中で元の位置にある要素が1つもない並び替えのことです。

例：
- A, B, C という参加者がいる場合
- A → B, B → C, C → A （全員が自分以外に当たる）
- A → A, B → C, C → B （Aが自分に当たるのでNG）

### アルゴリズム

1. Fisher-Yatesアルゴリズムで配列をシャッフル
2. 全員が自分以外に当たっているかチェック
3. 自分に当たっている人がいれば再シャッフル
4. 完全順列が見つかるまで繰り返し
